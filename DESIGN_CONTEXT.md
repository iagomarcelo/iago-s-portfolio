# Design Context - Iago Marcelo Portfolio

## Design Philosophy: Microsoft Enterprise Elegance

This portfolio follows a professional, corporate design approach inspired by Microsoft's design system. The aesthetic combines clarity, sophistication, and modern web design principles.

### Color Palette
- **Primary Blue**: #0078D4 (Microsoft Blue)
- **Teal Accent**: #00B7C3 (Interactive elements)
- **Light Gray**: #F3F3F3 (Backgrounds)
- **Dark Gray**: #333333 (Text)
- **White**: #FFFFFF (Cards, sections)

### Typography
- **Display/Headings**: Poppins Bold (48-64px for h1, 28-36px for h2)
- **Body**: Inter Regular (16px)
- **Captions**: Inter Regular (12-14px)

### Layout Principles
1. **Diagonal Dividers**: Angled separators between sections in blue gradient
2. **Floating Cards**: Cards with subtle shadows creating depth
3. **Asymmetric Layout**: 60/40 or 70/30 splits to avoid monotony
4. **Generous Whitespace**: 3rem-5rem margins between sections

### Animations
- **Entrance**: Fade-in + slide-up (300ms)
- **Hover**: Elevation (shadow increase) + color change
- **Transitions**: 300-400ms with cubic-bezier(0.4, 0, 0.2, 1)
- **Scroll**: Subtle parallax effects

### Key Components
- Hero section with diagonal background
- Floating technology cards
- Timeline for experience
- Project showcase with hover effects
- Contact section with floating elements

### Language Support
- Portuguese (PT-BR) as default
- English (EN) as secondary language
- Language toggle in header
